구글 SEO 최적화 블로그 스킨 수정 및 버튼 생성 팁

동영상 URL: https://www.youtube.com/watch?v=biO_5r7NVGI

동영상 개요

이 비디오는 구글 SEO(검색 엔진 최적화)에 더 적합한 블로그 스킨을 수정하는 방법과 HTML/CSS를 활용하여 버튼을 만드는 방법에 대한 실질적인 팁을 제공합니다.

주요 내용

  • 가독성을 위한 스킨 폭 조절:
    • 편집 디자인 지식에 따라 한 줄에 12어절 정도가 가독성이 좋다는 점을 언급하며, 티스토리 스킨의 본문 폭을 720 픽셀로 조절하는 방법을 설명합니다.
    • 이는 관리자 페이지의 스킨 편집 및 HTML 편집을 통해 CSS를 수정하여 이루어지며, 개발자 도구(F12)를 사용하여 '이너 인덱스' 태그의 폭을 확인하고 수정할 수 있습니다.
    • 폭을 조절하는 것이 구글 SEO 상위 노출에 유리하다는 주관적인 의견을 덧붙입니다.
  • 구글 SEO에 적합한 버튼 생성:
    • 이미지를 사용하는 대신 HTML 태그와 CSS를 이용하여 버튼을 만드는 것이 구글 SEO에 더 적합하다고 강조합니다.
    • 'CSS 버튼 제네레이터'를 통해 쉽게 버튼 코드를 생성하고, 색상(배경색, 호버 색상)을 변경하는 방법을 시연합니다.
    • 생성된 CSS 코드를 블로그의 CSS 파일에 추가하고, HTML 코드를 글쓰기 모드에서 사용하여 버튼을 삽입하며 원하는 링크 주소(href)와 버튼 텍스트를 지정하는 방법을 안내합니다.
    • 버튼 텍스트는 "여기를 클릭하세요"와 같은 모호한 문구 대신, 버튼을 클릭했을 때 얻을 수 있는 구체적인 효과나 효능을 짧게 쓰는 것이 좋다고 조언합니다.
  • 이미지 파일명 SEO 팁:
    • 많은 사람들이 이미지 파일명에 키워드를 넣어 SEO에 좋다고 생각하지만, 티스토리나 네이버 블로그처럼 CDN을 사용하는 플랫폼에서는 이미지가 업로드될 때 파일명이 자동으로 변경되므로 이러한 노력이 무의미할 수 있다고 언급합니다.
    • 다만 워드프레스와 같이 직접 블로그를 관리하는 경우에는 이미지 파일명 SEO가 유효할 수 있다고 덧붙입니다.

결론

이 비디오는 블로그 운영자들이 구글 SEO에 맞춰 블로그를 최적화하는 데 도움이 되는 실질적인 팁을 제공하며, 특히 개발자 관점에서 유용한 정보를 전달합니다. 스킨 폭 조절, HTML/CSS를 활용한 버튼 생성, 그리고 이미지 파일명 SEO에 대한 현실적인 조언을 통해 블로그의 검색 엔진 노출을 개선하는 데 기여합니다.
